Formularios (Resumen de Ejercicios)

1.

Formularios

Un formulario HTML es el mecanismo que usa una página web para recoger datos de un usuario y enviarlos a un servidor. Funciona como un contenedor (la etiqueta <form>) que agrupa diferentes controles (como <input>, <textarea>, etc.). Cuando el usuario pulsa el botón de enviar, el navegador empaqueta los datos de los controles (usando sus atributos name) y los envía al destino (atributo action) usando el método especificado (atributo method).

2.

GET/POST

Son los dos métodos principales para enviar datos de un formulario:

  • GET: Envía los datos añadiéndolos al final de la URL (ej: ...buscar.php?q=libros).
    • Ventajas: La URL se puede guardar en favoritos o compartir. Es bueno para búsquedas o filtros.
    • Desventajas: Es inseguro para datos sensibles (contraseñas) y tiene un límite de tamaño.
  • POST: Envía los datos de forma oculta en el "cuerpo" de la petición HTTP. La URL no cambia.
    • Ventajas: Es seguro para datos sensibles y no tiene límite de tamaño (ideal para subir archivos).
    • Desventajas: No se puede guardar en favoritos. Si recargas, el navegador pregunta si quieres "reenviar" los datos.

3.

Atributos

(de <form>)
accept-charset
Define la codificación de caracteres (ej: "UTF-8").
autocomplete
Activa ("on") o desactiva ("off") el autocompletado del navegador para el formulario.
enctype
Cómo se codifican los datos. Se usa multipart/form-data si se van a subir archivos.
novalidate
Si está presente, el navegador no validará los campos (required, pattern, etc.) antes de enviar.
target
Dónde mostrar la respuesta (ej: _self en la misma ventana, _blank en una pestaña nueva).

6.

Atributos button

Estos atributos (para <button> o <input type="submit">) sobrescriben los del <form>:

  1. formaction: Sobrescribe el action del formulario.
  2. formmethod: Sobrescribe el method (ej: GET o POST).
  3. formenctype: Sobrescribe el enctype (ej: para subir archivos).
  4. formnovalidate: Hace que ese botón envíe el formulario sin validar.
  5. formtarget: Sobrescribe el target (ej: _blank).
